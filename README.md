# 3D Portfolio

Interactive 3D portfolio website with Three.js animations, scroll-driven transitions, and project showcases.

## Features

- 3D animated hero section with Three.js
- Scroll-driven scene transitions
- Project gallery with interactive cards
- Responsive design for all screen sizes
- Performance-optimized 3D rendering

## Tech Stack

JavaScript, Three.js, HTML, CSS, GSAP

## Getting Started

### Prerequisites

- Node.js 18+
- npm

### Installation

```bash
git clone https://github.com/karthik-idikuda/karthik-3d-portfolio.git
cd karthik-3d-portfolio
npm install
```

### Usage

```bash
npm run dev
```
